Indicateurs delta visibles dans chaque cellule numérique :
  ↑ +23  (hausse vs mois précédent, en rouge si forte augmentation)
  ↓ -12  (baisse, en vert si le volume conservé diminue)

Filtres disponibles :
  - Agence (Toutes / H91358 / H92083)
  - Période (3 mois / 6 mois / 12 mois / personnalisé)

## C. GRAPHIQUES RECHARTS (optionnel, même page)
═══════════════════════════════════════════════════════

Deux graphiques sous le tableau :

1. Courbe "Volumes dans le temps"
   - Axe X : mois
   - Axe Y : importés / conservés (2 lignes)
   - Filtrable par agence

2. Barres empilées "Répartition par CDC"
   - Axe X : mois
   - Barres : particuliers + entreprises par CDC
   - Utile pour détecter une surcharge ou un déséquilibre

## D. NAVIGATION — DÉCISION VALIDÉE
═══════════════════════════════════════════════════════

Point d'entrée : section "Configs précédentes" dans la sidebar
de la page principale préterme-auto.

Chaque entrée = un cycle passé, avec :
  - le mois (ex: "Avril 2026")
  - le statut (✅ Terminé)
  - le nombre de clients conservés (ex: "412 clients")
  - un lien → "Voir le détail"

Le lien ouvre une page dédiée :
  /admin/preterme-auto/historique/[moisKey]

Cette page contient, dans l'ordre :
  1. Fiche synthèse du cycle (Bloc A — identité + seuils + volumes)
  2. Tableau historique tous cycles (Bloc B)
  3. Graphiques Recharts (Bloc C)

Bouton "← Retour" pour revenir à la page principale.
