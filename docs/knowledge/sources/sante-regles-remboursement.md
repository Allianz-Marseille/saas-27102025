# Santé : Règles de remboursement

## Principe du reste à charge

### Base de remboursement
- La Sécurité Sociale rembourse un pourcentage de la base de remboursement (BR)
- La complémentaire santé rembourse le reste (dans la limite des garanties souscrites)
- Le reste à charge = ce que le patient paie réellement

### Exemple concret
- Consultation médecin généraliste : 25€
- Base de remboursement SS : 25€
- Remboursement SS (70%) : 17,50€
- Remboursement complémentaire (30%) : 7,50€
- **Reste à charge : 0€** (si complémentaire 100%)

### Si dépassement d'honoraires
- Consultation médecin généraliste : 35€ (dépassement de 10€)
- Base de remboursement SS : 25€
- Remboursement SS (70%) : 17,50€
- Remboursement complémentaire (30% de 25€) : 7,50€
- **Reste à charge : 10€** (le dépassement n'est pas remboursé sauf garantie spécifique)

## Taux de remboursement Sécurité Sociale

### Soins courants
- Médecin généraliste : 70%
- Médecin spécialiste : 70%
- Kinésithérapeute : 60%
- Infirmier : 60%

### Soins dentaires
- Soins courants : 70%
- Prothèses dentaires : 70% (plafond)
- Orthodontie : 70% (plafond, généralement pour enfants)

### Soins optiques
- Verres correcteurs : 60% (plafond)
- Monture : forfait (généralement faible)
- Lentilles : 60% (plafond)

### Hospitalisation
- Frais d'hospitalisation : 80%
- Forfait hospitalier : non remboursé (sauf exceptions)

## Points de vigilance

### Dépassements d'honoraires
- Non remboursés par la Sécurité Sociale
- Remboursés par la complémentaire uniquement si garantie spécifique
- Vérifier les garanties de la complémentaire

### Plafonds de remboursement
- La Sécurité Sociale rembourse jusqu'à un plafond
- La complémentaire rembourse dans la limite des garanties
- Vérifier les plafonds selon les garanties

### Exclusions
- Certains soins ne sont pas remboursés (médecine douce, certains soins esthétiques)
- Vérifier les exclusions de la complémentaire
