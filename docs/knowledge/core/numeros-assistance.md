Numéros d'assistance disponibles pour les clients Allianz selon le domaine et le problème rencontré.

## Numéros d'assistance par domaine

- Plomberie
- Serrurerie
- Garde d'enfant

**Disponibilité** : France métropolitaine

## Utilisation dans les communications

### Pour les réponses aux clients

Lorsqu'un client rencontre un problème nécessitant une assistance :

1. **Identifier le type de problème** (habitation, auto/moto, banque)
2. **Fournir le numéro d'assistance correspondant**
3. **Préciser les horaires et conditions** si nécessaire
4. **Mentionner que le numéro est gratuit** pour l'assistance auto (0800)

### Format de présentation

**Habitation** :
> Pour une urgence habitation (plomberie, serrurerie, garde d'enfant), contactez le **01 40 25 52 95** (France métropolitaine).

**Auto/Moto** :
> En cas de panne ou crevaison, contactez l'assistance auto au **0800 103 105** (numéro gratuit, France métropolitaine).

**Banque** :
> En cas de perte ou vol de carte bancaire ou chéquier, contactez immédiatement le **0969 39 69 86** pour faire opposition (France métropolitaine).

### Points d'attention

- **Urgence banque** : En cas de perte/vol de carte, l'opposition doit être faite **immédiatement**
- **Assistance auto** : Numéro gratuit, disponible 24/7 généralement
- **Zone géographique** : Tous ces numéros sont valables en France métropolitaine uniquement
