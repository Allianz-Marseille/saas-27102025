# 13 - RÉGIME IRCEC 2026 (ARTISTES-AUTEURS)

## 1. INDEMNITÉS JOURNALIÈRES (IJ)
- **Règle :** Ils dépendent du Régime Général (CPAM) pour les IJ.
- **Condition :** Avoir cotisé sur une assiette suffisante.
- **IJ CPAM :** 50% du revenu journalier moyen (Plafond bas : **53,88 € / jour**).
- **Note :** Il n'y a pas de relais spécifique après 90 jours comme pour les libéraux.

## 2. INVALIDITÉ / DÉCÈS
- Couverture minimale via le régime de base. Les rentes sont calculées sur la moyenne des revenus, souvent fluctuants.

## 3. ANALYSE TECHNIQUE POUR BOB
- **Précarité :** C'est le régime le plus "pauvre" en termes de protection.
- **Argument Allianz :** Le contrat Prévoyance Allianz est souvent la SEULE véritable protection de l'artiste.
