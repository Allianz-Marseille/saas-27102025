# Gestion de droit commun — Sinistres

Source : Code des assurances, droit français. Quand les conventions (IRSA, IRSI) ne s'appliquent pas ou sont dépassées.

## Quand appliquer le droit commun

- Montant > seuils : IRSA > **6 500 € HT**, IRSI > **5 000 € HT**.
- Complexité : responsabilités multiples, contestation faits ou responsabilité.
- Demande d'expertise par l'assuré ou l'assureur.
- Sinistre hors convention : un seul véhicule ou non couvert IRSA/IRSI.

## Procédure type

1. Déclaration : 5 jours ouvrés (Code des assurances) ; vol : 2 j ouvrés ; CAT-NAT : 10 j après arrêté.
2. Expertise : désignation par l'assureur ou d'un commun accord ; en général sous 20 j ouvrés.
3. Indemnisation : 3 mois max après réception de toutes les pièces (sauf sinistres complexes).

## Obligations

- Assuré : déclaration en délai, préservation des preuves, coopération avec l'expert, fourniture des documents.
- Assureur : instruction dans des délais raisonnables, transparence garanties/exclusions, indemnisation conforme au contrat.

## Exclusions (Code des assurances, CGA)

- Faute intentionnelle de l'assuré ; défaut de déclaration dans les délais ; dommages non couverts.
