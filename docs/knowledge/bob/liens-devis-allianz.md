## Utilisation dans les communications

### Pour l'IA et les échanges avec les clients

Lorsqu'un client exprime un besoin d'assurance, l'IA doit :

1. **Identifier le type de besoin** du client
2. **Recommander le lien de devis approprié** (utiliser les liens de ce document avec code agence H91358)
3. **Fournir le lien cliquable** dans la réponse (format Markdown : `[Libellé](URL)`)
4. **Expliquer brièvement** ce que le client peut faire avec ce lien (devis gratuit, sans engagement, réponse sous 24-48h si pertinent)

### Format de présentation

**Exemple pour un devis auto** :
> Pour obtenir un devis personnalisé pour votre assurance auto : [Devis Auto](https://www.allianz.fr/forms/api/context/sharing/quotes/auto?codeAgence=H91358)
> Ce devis est gratuit, sans engagement et adapté à votre profil. Vous recevrez une réponse sous 24-48h.

**Exemple pour un devis habitation** :
> Pour votre assurance habitation : [Devis Habitation](https://www.allianz.fr/forms/api/context/sharing/fast-quotes/household?codeAgence=H91358)
> Le formulaire prend quelques minutes et vous permettra d'obtenir une estimation adaptée à votre logement.

**Exemple pour un devis santé / mutuelle** :
> Pour une complémentaire santé : [Devis Santé](https://www.allianz.fr/assurance-particulier/formulaire/devis-sante.html?codeAgence=H91358)
> Devis gratuit et sans engagement.

### Points d'attention

- **Tous les liens incluent le code agence H91358** pour que les devis soient attribués à l'agence.
- **Devis gratuits et sans engagement.**
- **Encourager le client** à compléter le formulaire en ligne pour une réponse rapide.
- **Proposer un rendez-vous** (lien « Formulaire de contact (rendez-vous) ») si le besoin est complexe ou si le client préfère un accompagnement personnalisé.
