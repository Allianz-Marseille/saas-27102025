## 2. Cases "Revenu" pour calcul IJ
* **Libéral (2035) :** Bénéfice (Case **CP**) + Cotisations sociales personnelles (Case **BT**).
* **Commerçant (2033) :** Résultat fiscal (2031 Case **1**) + Cotisations sociales (2033-D Case **380**).
* **Gérant Majoritaire (IS) :** Ne pas prendre le bénéfice société. Utiliser la rémunération nette versée (2033-D ou 2058-C). Vérifier si les dividendes sont à assurer (souvent exclus).
* **Auto-entrepreneur :** CA brut - abattement forfaitaire (Vente: 71% / Presta BIC: 50% / Libéral BNC: 34%).

## 3. Points de vigilance
- **Réintégration Madelin :** Toujours réintégrer les cotisations Madelin pour obtenir le revenu réel avant prévoyance.
- **Vérification Dividendes :** Si le dirigeant (IS) se paie majoritairement en dividendes, alerter sur le manque de couverture en cas d'arrêt de travail.
- **Source :** Code Général des Impôts, URSSAF 2026.
