# Fichier audio pour la célébration du Nouvel An

## Instructions

Pour que l'effet Star Wars du Nouvel An fonctionne avec le son, vous devez ajouter un fichier audio dans ce dossier.

**Nom du fichier requis** : `new-year-celebration.mp3`

## Recommandations

- **Format** : MP3 (recommandé pour la compatibilité)
- **Durée** : 5-10 secondes (son de fête/célébration)
- **Volume** : Le volume sera ajusté automatiquement à 60% dans le code

## Sources de sons libres de droits

Vous pouvez trouver des sons de fête/célébration sur :
- [Freesound.org](https://freesound.org) - Rechercher "celebration", "party", "new year"
- [Zapsplat](https://www.zapsplat.com) - Sons gratuits avec compte
- [Mixkit](https://mixkit.co/free-sound-effects/) - Sons gratuits

## Alternative

Si vous ne souhaitez pas ajouter de fichier audio, le composant continuera de fonctionner sans son (les erreurs sont gérées silencieusement).
