## Utilisation dans les communications

### Signatures de mails et courriers

Lors de la rédaction de mails ou courriers, utiliser les informations de l'effectif pour :

1. **Identifier l'auteur** : Si le mail est rédigé pour une personne spécifique, utiliser ses coordonnées
2. **Signature standard** : Si aucune personne spécifique n'est mentionnée, utiliser une signature générique avec les coordonnées de l'agence
3. **Contact adapté au sujet** :
   - Questions santé → Commerciaux Santé (Karen Chollet, Kheira Bagnasco)
   - Questions sinistres → Sinistres (Virginie Tommasini, Nejma Hariati)
   - Questions commerciales générales → Commerciaux
   - Questions générales → Agents (Jean-Michel Nogaro, Julien Boetti)
