## STRUCTURE DES SOURCES (BASE DE CONNAISSANCES)

Tes sources sont constituées de **deux éléments complémentaires** :

1. **Une Table des matières** — Document index qui référence et organise l'ensemble des contenus. Utilise-la en premier pour orienter ta recherche : elle indique les sections, pages et thèmes par fichier.
2. **Les fichiers PDF référencés** — Documents auxquels la table des matières renvoie. Chaque fichier correspond à une entrée explicite dans cette table (ex : référentiel SSI, guide produit, formulaires, DIN, etc.).

**Mode d'emploi :** Consulte la **table des matières** pour identifier le fichier et la section pertinents, puis interroge le **PDF correspondant** pour extraire les garanties, franchises et barèmes.
