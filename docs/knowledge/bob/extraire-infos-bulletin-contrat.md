# Extraire les infos d’un bulletin ou d’un contrat

Fiche de connaissance pour **guider le conseiller** qui souhaite extraire des informations d’un bulletin de salaire, d’un contrat de prévoyance/santé ou d’une liasse (ex. 2035). Bob peut analyser un document uploadé et en tirer les points clés.

---

## 1. Comment utiliser Bob pour extraire

- **Upload** : le conseiller peut joindre un fichier (PDF, image, texte) à sa question dans l’interface Bob.
- **Demande claire** : formuler la demande en précisant ce que l’on veut (ex. « Extraire les montants IJ et la base de calcul », « Résumer les garanties du contrat », « Lister les lignes santé de cette fiche de paie »).
- **Contexte** : indiquer le type de document (bulletin de paie, attestation, contrat groupe, 2035 BNC, etc.) pour que Bob applique le bon cadre (fiche paie, DUE, 2035, etc.).

---

## 2. Infos utiles à extraire selon le document

| Type de document | Infos utiles à extraire |
|------------------|--------------------------|
| **Bulletin de paie** | Base de cotisation prévoyance/mutuelle, part employeur/salarié, tranches A/B/C, libellés des lignes santé. Voir `fiche-paie-sante.md`. |
| **Contrat prévoyance / mutuelle** | Franchises (IJ), seuils invalidité, capital décès, exclusions, délais de carence. |
| **2035 (BNC)** | Revenus, charges, assiette IJ, éléments pour calcul des frais généraux. Voir `2035-bilan-tns.md`. |
| **Attestation mutuelle** | Période de validité, organisme, numéro d’affiliation, niveau de garantie si indiqué. Voir `attestation-mutuelle.md`. |
| **DUE / offre groupe** | Garanties proposées, taux, bases, population couverte. Voir `due-contrat-groupe.md`. |

---

## 3. Réponse type de Bob

- Structurer la réponse en **liste ou tableau** (montants, dates, garanties).
- **Citer la source** : « D’après le document fourni… » et indiquer la fiche de connaissance utilisée pour le décodage (ex. *Sources : fiche-paie-sante.md*).
- Proposer une **action suivante** : rédiger un mail client, comparer avec un autre contrat, intégrer dans un bilan TNS, etc.
