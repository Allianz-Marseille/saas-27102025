# Visages / photos des agents IA

Dépose ici les images (avatars, visages, photos) qui représentent chaque agent IA.

**Structure recommandée :**

```
public/agents-ia/
  m-plus-3/
    avatar.png
  preterme-auto/
    avatar.png
  preterme-ird/
    avatar.png
  …
```

**Usage dans l’app :**  
URL = `/agents-ia/<slug>/avatar.png`  
Ex. : `/agents-ia/m-plus-3/avatar.png`

Voir aussi `docs/agents-ia/README.md` pour la documentation des agents (rôles, mapping).
